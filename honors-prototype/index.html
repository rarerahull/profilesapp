Project Summary

Student: Rahul Sharma
Course: CSCE 3420 – Internet Programming
Faculty Mentor: Dr. Ryan Garlick

This UI is the static front-end prototype for the "Eagle Park Finder" application. Later phases will add JavaScript-based interactivity and an AWS serverless back end that simulates real-time parking availability.

How it will work
  • Front-end built with HTML, CSS, Bootstrap, and JavaScript.
  • Interactive map view + lot list with live availability.
  • Data served from a Node.js function on AWS using a NoSQL database (simulated sensor data).
  • Students can quickly scan which lots have open spaces before they reach campus.
